Ce projet vise à développer un système intelligent capable d’aider un centre de formation à mieux comprendre ses apprenants, améliorer leurs parcours et optimiser l’organisation interne grâce à des techniques de Machine Learning.

Le dataset utilisé décrit plusieurs aspects des apprenants :

Données personnelles : âge, région, genre

Données académiques : cours suivis, notes en Java/Python/Cloud…, absences

Niveau atteint : Level 1, 2, 3…

Soft skills : communication, créativité, travail en équipe

Satisfaction : avis et feedback

L’objectif global est de prédire la réussite, segmenter les profils et recommander des actions personnalisées permettant d’améliorer l’expérience des apprenants.
